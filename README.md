# Pong-game  

In this game, you have to hit the ball with your pong to win the game.  
If ball touches the wall beside your pong opponent will get a point.  
Player with more points will win the game.  

Player's 1 movement : 

Keys   | Direction 
|:---:|:---:|
|  w | Up | 
| s | Down | 

Player's 2 movement : 

Keys   | Direction 
|:---:|:---:|
|  Up arrow key | Up | 
| Down arrow key | Down |
